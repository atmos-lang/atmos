# at-tag : `parser_at` accepts a bare `@:tag`

## Goal

Let the `@`-qualifier accept a bare tag (`@:x`), not only
`@(e)`, `@id`, `@num`.
`parser_at` is shared, so this enables the same form at every
`@` site.

```
emit @:global (e)       ;; was: emit @(:global) (e)
spawn @:x T()           ;; was: spawn @(:x) T()
[@:x = v]               ;; was: [@(:x) = v]
t@:x                    ;; was: t.x only
```

## Scope

Target sugar only.
The payload / call after the target is unchanged and still
needs its own parentheses (`emit @:global (T())`, not
`emit @:global T()`).
